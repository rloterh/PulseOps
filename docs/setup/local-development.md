# Local Development

## Prerequisites

- Node.js 22 or newer
- Corepack enabled

## Bootstrapping

1. Run `corepack pnpm install`.
2. Copy `.env.example` to `.env.local`.
3. Start the app with `corepack pnpm dev`.

## Verification

Run `corepack pnpm check` to execute linting, type-checking, tests, and builds across the workspace.
